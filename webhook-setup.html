💳 Stripe Webhook Configuration

Configure seu Stripe para enviar eventos de pagamento para o sistema.

📋 Step 1: Get your webhook URL

https://node00-omega.onrender.com/webhook/stripe

⚙️ Step 2: Configure in Stripe Dashboard

1. Acesse Stripe Dashboard → Developers → Webhooks

2. Clique em "Add endpoint"

3. Cole a URL acima

4. Selecione os eventos:

checkout.session.completed
customer.subscription.created
customer.subscription.updated

5. Clique em "Add endpoint"

🔐 Step 3: Add Signing Secret to .env

Após criar o webhook, copie o "Signing secret" e adicione ao arquivo .env:

STRIPE_WEBHOOK_SECRET=whsec_xxx

✅ Step 4: Test the webhook

No Stripe Dashboard, use o botão "Send test webhook" para verificar.

Eventos reais aparecerão em WEBHOOK-LOG.txt
